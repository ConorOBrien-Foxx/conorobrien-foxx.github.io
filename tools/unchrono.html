Unchrono

Stopwatch Timings to Absolute Timestamps

How to Use

Convert a list of stopwatch timings (aka time per lap) to absolute timestamps given the wall clock time at which the timer was stopped. Please use YYYY-MM-DD HH:MM:SS for the wall clock reading, and DD:HH:MM:SS, HH:MM:SS, MM:SS, or simply SS formats for stopwatching timings.

Press and hold the Now button to constantly update the time, and release when satisfied with the time.

Note: This app uses the Temporal API, which is not universally supported. Works best on Firefox.
